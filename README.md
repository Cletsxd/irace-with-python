# IRACE
## Example for Python programs

# Requirements

- Python 3.6
- R 4.02
- IRACE (R package) 3.4

## Python Packages

- argparse
- logging
- sys

## Instrutions

- Move to ProjectFolder/irace/
- How to run this on windows:
	'irace --target-runner target-runner.bat'
- On linux run the commando:
	'irace'
- Done
